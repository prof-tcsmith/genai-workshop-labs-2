# Northwind Cloud — Support Knowledge Base

## Accounts & sign-in
- Reset your password from the sign-in page using "Forgot password"; the reset link expires after 30 minutes.
- Accounts lock after 5 failed sign-in attempts and unlock automatically after 15 minutes.
- Two-factor authentication can be enabled under Settings → Security.

## Billing
- Plans are billed monthly on the date you first subscribed.
- You can change plans at any time; upgrades are prorated, downgrades take effect next cycle.
- Invoices are available under Settings → Billing → Invoices.

## Features
- Workspaces let teams share dashboards; the Free plan allows 1 workspace, Pro allows 10.
- Data export is available in CSV and JSON from any dashboard's "..." menu.
- API access requires a Pro plan; generate API keys under Settings → Developer.

## Troubleshooting
- If dashboards won't load, first clear the browser cache and retry in a private window.
- The "export failed" error is usually a timeout on very large datasets; export a narrower date range.
- Service status is published at status.northwind.example.
