Enterprise AI for IS Faculty

A hands-on workshop. You build a GenAI system one layer at a time — then watch those same blocks become a real application that turns your course materials into Canvas-ready quizzes. Everything runs in your browser: no install, no account.

Start here: when an app asks for a participant code (and, in a couple of labs, an OpenAI key), Tim hands it out at the session. Keys stay server-side or in your browser only — nothing is stored. Free apps sleep when idle, so give one a few seconds to wake on first click.
